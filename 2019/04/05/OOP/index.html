一、面向对象编程

面向对象编程（Object-oriented programming,OOP）是一种程序设计泛型，将面向对象作为程序的羁绊单元，将程序和数据封装在其中，提高重用性、灵活性和扩展性。

第二章：写的都是看到的 —— 面向对象编程

（一）、两种编程风格

面向对象编程：将需求抽象成一个对象，针对这个对象分析其特征（属性）与动作（方法）。特点：封装，继承，多态；

（二）、包装明星 —— 封装

创建一个类：1、声明一个函数保存在变量里，首字母大写（Fun）；通过 this 变量添加属性或者方法。

通过 this 添的属性，方法是在当前对象上添加的，通过类创建新对象时，this指向的属性和方法都会得到相应的创建。然而 js 是基于原型的语言，所以每创建一个对象（包括函数）时，都会有一个原型 prototype 用于指向其继承的属性、方法。通过 prototype 继承的属性和方法是每个对象通过 prototype 访问到的，所以每次通过类创建一个新对象是这些属性和方法不会再创建。
constructor 是一个属性，在 prototype （原型对象中）用于指向拥有整个原型对象的函数或对象。

面向对象：一些属性的隐藏与暴露；私有属性、方法；共有属性、方法；保护方法等。
