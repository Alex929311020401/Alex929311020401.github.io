jQuery API

选择器

一、基本选择器

元字符 使用 \\ 转义
语法 说明
#id id
.class class
* 通配
selector1,selector2,selectorN 每个选择器匹配到的元素合并后一起返回

二、层级

语法 说明
ancestor descendant 给定的祖先元素下匹配所有的后代元素
parent > child 给定的父元素下匹配所有子元素
prev + next 紧邻在 prev 元素后的 next 元素
prev ~ siblings prev 后的所有兄弟元素

三、基本筛选器

语法 说明
:first 第一个元素
:not(selector) 排除给定的选择器匹配的元素
:even 匹配所有索引值是 偶数 的元素 idx=0，2，4…
:odd 匹配所有索引值是 奇数 的元素 idx=1,3,5…
:eq(index) 匹配指定 索引值 的元素
:gt(index) 匹配所有大于 给定索引值的元素
:lang 匹配指定语言的所有元素;eg: <div lang= “en”> 1.9+
:last 给定的父元素下匹配所有子元素
:lt(index) 给定的父元素下匹配所有子元素
:header 紧邻在 prev 元素后的 next 元素
:animated prev 后的所有兄弟元素
:focus prev 后的所有兄弟元素 1.6+
:root prev 后的所有兄弟元素 1.9+
:target prev 后的所有兄弟元素 1.9+

四、内容

| :contains(text) | 匹配包含文本的元素 |
| :empty | 不包含子元素或者文本的空元素 |
| :has(selector) | 匹配含有选择器所匹配的元素的元素 |
| :parent | 匹配含有子元素或者文本的元素 |

五、可见性

| :hidden | 匹配所有不可见元素，或者type为”hidden“的元素 |
| :visible | 匹配所有可见元素 |

六、属性

| [attribute] | 匹配包含给定属性的元素 |
| [attribute = value] | 匹配给定属性是某个特定值的元素 |
| [attribute != value] | 匹配不包含给定属性，或属性不等于某个特定值的元素 |
| [attribute ^= value] | 匹配给定属性以某些值开始的元素 |
| [attribute $= value] | 匹配给定属性以某些值开始的元素 |
| [attribute *= value] | 匹配给定属性以某些值开始的元素 |
| [attrSel1[attrSel1[attrSel1] | 匹配给定属性以某些值开始的元素 |

七、子元素

| :first-child | 匹配 |
| :last-child | 匹配 |
| :nth-child | 匹配 |
| :nth-last-child | 匹配 |

| :first-of-type | 匹配 |
| :last-of-type | 匹配 |
| :nth-last-child | 匹配 |
| :nth-last-of-type | 匹配 |

| :nth-last-child | 匹配 |
| :first | 匹配 |
| :first | 匹配 |
